# Titanic 🚢

## Sobre o Projeto
Projeto de Análise Exploratória de Dados (EDA) e Machine Learning aplicado ao dataset do Titanic, disponibilizado pelo Kaggle na competição [Titanic - Machine Learning from Disaster](https://www.kaggle.com/competitions/titanic).
